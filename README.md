# Netflix-Content-Insights-Analysis

## 📌 Overview
This project analyzes Netflix's dataset to uncover trends in movies and TV shows using Python. The analysis focuses on content distribution, genre trends, and viewing patterns.

## 🎯 Objectives
- Analyze Movies vs TV Shows distribution  
- Identify popular genres  
- Explore content trends over time  
- Understand ratings and duration patterns  

## 🧠 EDA Approach
- Data cleaning and preprocessing  
- Exploratory Data Analysis (EDA)  
- Data visualization  
- Outlier detection  

## 🛠️ Tech Stack
- Python  
- Pandas  
- Matplotlib / Seaborn  

## 📂 Dataset
- Netflix Titles Dataset  
- Included in this repository
